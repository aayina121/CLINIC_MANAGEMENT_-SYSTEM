# CLINIC_MANAGEMENT_SYSTEM
A Low-cost way to manage patient records, appointments, billing, and clinic operations using Microsoft Excel. It’s especially useful for small clinics that don’t need full-scale hospital software.
The project is organized across four dedicated Excel worksheets:
 (1) Patient Records — stores complete information for 25 patients including name, age, assigned doctor, visit fee, number of visits, total bill, cost status (HIGH COST / NORMAL), fee type (EXPENSIVE / AFFORDABLE), follow-up flag, and attendance percentage.
 (2) Visit Tracker — records daily attendance (P/A) for each patient across a 5-day working week (Monday to Friday) and auto-calculates total visits and attendance percentage. 
 (3) Dashboard — provides a real-time summary including total patients, revenue, average billing, doctor-wise breakdown, and cost distribution.
 (4) Formulas Used — a reference sheet documenting all Excel formulas used in the project.
